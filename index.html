SYSTEM BRIEFING: ERROR PROTOCOLS

Analyze the four main bug types before beginning the simulation:

Error Type Definition Example
Syntax Error Broken grammar. The computer cannot translate the code. if x == 5 (missing colon)
Name Error Referencing a variable or function that doesn't exist. print(total) (if total isn't defined)
Type Error Performing an operation on the wrong data type. "High Score: " + 10 (string + int)
Logic Error The code runs, but the output is not what you intended. area = w + h (should be multiplication)
